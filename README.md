# MERN_SocksShop

It is a MERN e-commerce project build through udemy course with <a href='https://github.com/bradtraversy'>Brad Traversy</a>. 

I used it to learn the basics of Node.js - Express framework and MongoDB  implementation along with Redux practice. 
I would highly recommend it to anyone looking for a concise and well explained mid-level MERN project.
